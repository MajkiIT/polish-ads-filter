Jak zainstalować Nano Defender na Firefoksie, Waterfoksie, (Chr)Operze? albo Chrome?

Spis treści

  1. FIREFOX QUANTUM (57+)

  2. WATERFOX

  3. (CHR)OPERA

  4. GOOGLE CHROME/CHROMIUM/VIVALDI

I. FIREFOX QUANTUM (57+)

  1. Zainstaluj Nano Defender (kliknij na plik xpi, zezwól, a następnie dodaj).

  2. Jeśli używasz Nano Adblockera, to lista filtrów Nano Defender Integration powinna się automatycznie po chwili aktywować.

  3. Jeśli używasz uBlocka Origin wykonaj dodatkowe kroki:

    • Włącz listę filtrów Adblock Warning Removal List w zakładce Zewnętrzne filtry

    • Subskrybuj listę filtrów Nano Defender Integration: https://gitcdn.xyz/repo/NanoAdblocker/NanoFilters/master/NanoMirror/NanoDefender.txt

    • Wejdź do panelu sterowania uBlocka, zaznacz Jestem zaawansowanym użytkownikiem w karcie Ustawienia. Następnie kliknij na zębatkę, która się pojawiła obok tej opcji i zmień wartość unset dla wpisu userResourcesLocation na: https://gitcdn.xyz/repo/NanoAdblocker/NanoFilters/master/NanoFilters/NanoResources.txt

      Uwaga: Jeśli już wykorzystujesz opcję User Resources, to zamień uBlock Origin na Nano Adblocker lub ręcznie połącz (scal) zasoby z uBlock Origin

    • Dodaj listę filtrów Nano filters: https://gitcdn.xyz/repo/NanoAdblocker/NanoFilters/master/NanoFilters/NanoBase.txt

    • Dodaj listę filtrów Nano filters - Whitelist https://raw.githubusercontent.com/NanoAdblocker/NanoFilters/master/NanoFilters/NanoWhitelist.txt

  4. Ciesz się z działającego Nano Defendera!

II. WATERFOX

    1. Zainstaluj najnowszy plik xpi Classic Add-ons Archive, kliknij Dodaj do Firefoksa, a następnie Dodaj.

    2. Kliknij na ikonę Classic Addons Archive, a potem wklej do paska adresu caa:addon/checkcompatibility2, kliknij Install now, a następnie Dodaj.

    3. Zainstaluj Nano Defender (kliknij na plik xpi, zezwól, a następnie dodaj).

    4. Jeśli używasz Nano Adblockera, to lista filtrów Nano Defender Integration powinna się automatycznie po chwili aktywować.

        Uwaga: Jeśli już wykorzystujesz opcję User Resources, to zamień uBlock Origin na Nano Adblocker lub ręcznie połącz (scal) zasoby z uBlock Origin

III. (CHR)OPERA

(wersja wideo - www.youtube.com/watch?v=R5iB1-6B1A0)

  1. Zainstaluj Extension Source Viewer.

  2. Wejdź na chrome.google.com/webstore/detail/nano-defender/ggolfgbegefeeoocgjbmkembbncoadlb i kliknij na pasku adresu na ikonkę CRX, a następnie kliknij na install.

  3. Wejdź na opera://extensions, a następnie kliknij zainstaluj przy Nano Defender, potwierdź jeszcze raz naciskając zainstaluj.

  4. Jeśli używasz Nano Adblockera, to lista Nano Defender Integration powinna się automatycznie po chwili aktywować - opcjonalnie sprawdź i aktywuj.
